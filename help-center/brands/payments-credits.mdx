# Como funcionam os créditos na Conty

A Conty funciona com um sistema de créditos pré-pagos. Você adiciona saldo antes de criar campanhas, e esse saldo é consumido conforme os conteúdos são aprovados.

<Card title="Créditos pré-pagos" icon="wallet">
  Adicione créditos antes de criar campanhas. O saldo fica disponível para uso imediato.
</Card>

<Card title="Mínimo de R$1.500" icon="coins">

## Regras importantes sobre créditos

<Warning>
  Os créditos são adicionados **up-front** (antecipadamente). Não é possível criar campanhas sem saldo disponível.
</Warning>

<AccordionGroup>
  <Accordion title="Os créditos expiram?">
    Os créditos têm validade de **12 meses** a partir da data de compra. Após esse período, o saldo não utilizado expira.
  </Accordion>
  <Accordion title="Os créditos são reembolsáveis?">
    **Não.** Os créditos não são reembolsáveis após a compra.
    
    A única exceção é quando **não foi encontrado nenhum criador** para sua campanha. Nesse caso, o valor reservado é devolvido como crédito.
  </Accordion>
  <Accordion title="Quando o saldo é consumido?">
    O saldo da campanha só diminui **na aprovação dos conteúdos**. Isso significa que:
    - Você pode recusar vídeos que não gostar sem perder créditos
    - Só paga pelos conteúdos que realmente aprova
    - O valor fica reservado até a aprovação ou recusa
  </Accordion>
  <Accordion title="Posso usar os créditos em várias campanhas?">
    Sim. Os créditos ficam na sua conta e podem ser usados para criar quantas campanhas quiser, enquanto houver saldo disponível.
  </Accordion>
</AccordionGroup>

## Como adicionar créditos

<Steps>
  <Step title="Acesse o dashboard">
    Faça login em [dashboard.appconty.com](https://dashboard.appconty.com/).
  </Step>
  <Step title="Vá para a área de pagamentos">
    No menu, acesse a seção de créditos ou pagamentos.
  </Step>
  <Step title="Escolha o valor">
    Selecione o valor que deseja adicionar (mínimo R$1.500).
  </Step>
  <Step title="Realize o pagamento">
    Finalize o pagamento pelo método disponível.
  </Step>
  <Step title="Créditos disponíveis">
    Após a confirmação, os créditos estarão disponíveis para uso imediato.
  </Step>
</Steps>

## Resumo das regras

| Regra | Detalhe |
|-------|---------|
| Valor mínimo | R$1.500 |
| Validade | 12 meses |
| Reembolso | Não (exceto se não houver criadores) |
| Consumo | Apenas na aprovação do conteúdo |
| Uso | Múltiplas campanhas |

## Dúvidas comuns

<AccordionGroup>
  <Accordion title="Por que o mínimo é R$1.500?">
    O valor mínimo garante que sua marca tenha saldo suficiente para campanhas relevantes e mantém a liquidez do marketplace, beneficiando tanto marcas quanto criadores.
  </Accordion>
  <Accordion title="Posso adicionar mais créditos depois?">
    Sim. Você pode adicionar créditos a qualquer momento, sempre respeitando o valor mínimo de R$1.500 por transação.
  </Accordion>
  <Accordion title="Como acompanho meu saldo?">
    O saldo disponível aparece no dashboard. Você também pode ver o histórico de consumo por campanha.
  </Accordion>
  <Accordion title="E se meus créditos estiverem acabando durante uma campanha?">
    Você receberá alertas quando o saldo estiver baixo. Adicione mais créditos para manter suas campanhas ativas.
  </Accordion>
</AccordionGroup>
