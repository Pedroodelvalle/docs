## Novas funcionalidades

- Sistema completo de campanhas para marcas
- Portal do criador para participar de campanhas
- Sistema de pagamentos integrado
- Dashboard de acompanhamento de resultados
- Sistema de revisão e aprovação de entregas

## Para criadores

- Cadastro e perfil profissional
- Visualização de campanhas disponíveis
- Upload de vídeos
- Acompanhamento de status de campanhas
- Histórico de pagamentos

## Para marcas

- Criação e gerenciamento de campanhas
- Briefing detalhado para criadores
- Revisão e aprovação de vídeos
- Sistema de créditos e pagamentos
- Dashboard de resultados e métricas
</Update>

## Como acompanhar

As atualizações são organizadas por versão, com as mais recentes primeiro. Cada lançamento inclui:

- **Novas funcionalidades** - O que foi adicionado
- **Melhorias** - O que foi aprimorado  
- **Correções** - Bugs que foram resolvidos
- **Notas técnicas** - Mudanças para desenvolvedores
